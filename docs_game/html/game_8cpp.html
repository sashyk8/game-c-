Проверяет столкновение осей параллелепипедов.

Аргументы
p1xX позиция первого параллелепипеда.
p1yY позиция первого параллелепипеда.
hs1xПоловина размера по X первого параллелепипеда.
hs1yПоловина размера по Y первого параллелепипеда.
p2xX позиция второго параллелепипеда.
p2yY позиция второго параллелепипеда.
hs2xПоловина размера по X второго параллелепипеда.
hs2yПоловина размера по Y второго параллелепипеда.
Возвращает
true если параллелепипеды сталкиваются.
false если параллелепипеды не сталкиваются.
